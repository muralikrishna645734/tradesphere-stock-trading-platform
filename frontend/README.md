# TradeSphere 🚀

A full-stack stock trading platform inspired by Zerodha.

## Features

- Responsive trading dashboard
- Holdings and positions management
- Portfolio analytics
- React Router navigation
- REST API backend
- MongoDB integration
- Interactive charts

## Tech Stack

### Frontend
- React.js
- Bootstrap
- React Router DOM

### Dashboard
- React.js
- Chart.js
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Folder Structure

```bash
TradeSphere/
│
├── frontend/
├── dashboard/
├── backend/
```

## Installation

### Frontend

```bash
cd frontend
npm install
npm start
```

### Dashboard

```bash
cd dashboard
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm start
```

## Environment Variables

Create `.env` inside backend:

```env
MONGO_URL=your_mongodb_connection_string
PORT=3002
```
